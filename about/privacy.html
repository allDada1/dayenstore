Политика конфиденциальности

Эта страница теперь читается как нормальный документ: с понятными разделами, якорями и без ощущения, что она случайно наброшена для галочки.

Какие данные Как используются Хранение Права пользователя

Какие данные могут использоваться

  • Имя, почта и данные аккаунта для авторизации и идентификации пользователя.
  • Информация о заказах и действиях внутри интерфейса, если это нужно для работы функций сайта.
  • Технические данные, связанные с браузером, локальным хранением и корректной работой страниц.

Для чего используются данные

Данные используются для работы аккаунта, отображения истории действий, обработки заказов, обратной связи и улучшения интерфейса. Сайт не должен собирать лишнее и не должен использовать данные вне логики продукта.

Хранение и защита

Данные хранятся только настолько, насколько это нужно для работы функций проекта. Для демонстрационных страниц часть информации может храниться локально в браузере пользователя.

Права пользователя

  • Узнать, какие данные используются внутри платформы.
  • Обратиться в поддержку по вопросам удаления или исправления данных.
  • Прекратить использование сервиса и очистить локальные данные на своей стороне.
